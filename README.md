

# ComNetAppDesign

### 计算机网络应用设计实验——网上书店



#### 老师要求

> 应用程序要求实现一个网上书店的基本功能，要求具有前端（用户）和后端（管理）功能，基于Web运行方式。前端具有浏览书目、购物车等功能，后端具有管理书目、基本统计功能。
>
> - 客户端操作系统为Windows



#### 开发环境

- 前端框架：Bootstrap
- 前端开发语言：HTML5   CSS    JavaScript
- 后端框架：Django
- 后端开发语言：Python
